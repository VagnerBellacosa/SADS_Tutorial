Informativos

Introdução

O SADS/SS fornece uma grande variedade de consultas e referências cruzadas relativas aos cadastramentos de usuários e suas habilitações que foram efetuados em sua base de dados.

O Sistema de Segurança fornece também consultas referentes às funções de controle por ele efetuadas, tais como: bloqueio de estações, desabilitação de transações, limitação de horário de uso de aplicativos, etc.

Outro recurso oferecido pelo SADS/SS é a auditoria no cadastro de usuários e habilitações. Através dela, toda manutenção efetuada nos dados cadastrais dos usuários e em suas habilitações, pode posteriormente ser consultada e impressa. Descobre-se, assim, qual a atualização efetuada, qual o usuário agente da operação e qual o usuário afetado pela operação.

O Sistema de Segurança oferece também vários informativos impressos, contendo informações diversas a respeito dos cadastramentos de usuários e habilitações.

Estes relatórios são emitidos por jobs que podem ser disparados através de transações on-line, cadastradas no menu de operação "ROTEMI".

Todas as consultas, referências cruzadas e auditorias, suportadas pelo Sistema de Segurança, serão detalhadamente explicadas a seguir.

Serão descritos os títulos dos campos apresentados em cada consulta, sendo que para tomar conhecimento dos valores que estes campos podem assumir, deve-se consultar o item "Usuários e habilitações", nesta mesma seção.

Os relatórios não serão descritos nesta seção por estarem detalhadamente explicados na Seção III, "Funções de apoio", item "Utilitários batch".

Para consultar os informativos do Sistema de Segurança, deve-se utilizar o menu de escolha "INFSEG".

Consultas gerais

Para efetuar qualquer consulta no Sistema de Segurança, deve-se utilizar o menu de operação "CONGERSEG". Nele pode-se disparar as consultas relacionadas a seguir.

exista alguma limitação.

Usuários a liberar

Sempre que são incluídas ou alteradas as habilitações de aplicativos, transações ou elementos de um usuário, deve-se liberá-lo a fim de que suas novas habilitações passem a ter validade para os testes efetuados pelo Sistema de Segurança e a fim de que o usuário possa efetuar logon.

Para tanto, o SADS/SS disponibiliza uma consulta a usuários que necessitam ser liberados, possibilitanto ao Administrador de Segurança fazê-lo.

Argumentos da consulta

Transações desabilitadas

Esta consulta exibe as transações que estão correntemente desabilitadas, para facilitar o controle do administrador de segurança quanto às suas habilitações.

Uma transação desabilitada não pode ser processada por quaisquer que seja o usuário.

As transações podem ser desabilitadas manualmente por usuários autorizados ou automaticamente pelo próprio host. A desabilitação automática pelo host ocorre após um certo número de cancelamentos da transação, podendo este número ser parametrizado na instalação.

Quando uma transação é desabilitada manualmente, pode-se fazê-la por um certo período, após o qual a transação será automaticamente habilitada pelo Sistema de Segurança.

Associado a uma desabilitação, existirá cadastrado o motivo pelo qual a transação se encontra neste estado.

Argumentos da consulta

Para utilizar a consulta às transações desabilitadas, deve-se informar o seguinte argumento no menu de operação "CONGERSEG":

Operacao - "TRA".

Informações apresentadas na consulta

Para cada uma das transações desabilitadas, serão informados:

Menu - Código do menu referente à transação desabilitada (sssmm).

Opcao - Código da opção referente à transação desabilitada (nn).

Data Final - Data final da desabilitação, após a qual a transação será habilitada.

Hora Final - Hora final da desabilitação, após a qual a transação será habilitada.

Resp. - Número do usuário responsável pela desabilitação da transação.

Motivo - Motivo da desabilitação.

Estações bloqueadas

As estações podem ser bloqueadas automaticamente, quando se detecta, no momento do logon, uma tentativa de burlar a segurança, ou podem ser bloqueadas manualmente, através de uma transação on-line no Sistema de Segurança.

Para auxiliar no controle das estações bloqueadas, é oferecida tal consulta pelo Sistema de Segurança.

Argumentos da consulta

Para utilizar a consulta às estações bloqueadas, deve-se informar o seguinte argumento no menu de operação "CONGERSEG":

Operacao - "BLO ".

Informações apresentadas na consulta

Códigos de todas as estações correntemente bloqueadas.

Para efetuar referências cruzadas, deve-se utilizar o menu de escolha "REFCRUSEG".

Usuários por elemento

Esta consulta apresenta todos os usuários habilitados a trabalhar com um elemento, ou grupo de elementos, em um aplicativo ou lista de elementos específica.

Argumentos da consulta

Para utilizar a consulta usuários por elemento, deve-se informar os seguintes argumentos no menu de operação "USUPORELE":

Aplicativo/Lista de Elementos - Código do aplicativo ou lista de elementos para o qual o elemento está habilitado.

Empresa - Código da empresa na qual o aplicativo/lista de elementos está cadastrado e na qual os usuários podem ter acesso ao elemento (opcional).

Se não informado, a consulta será feita para todas as empresas.

Estabelec - Código do estabelecimento no qual o aplicativo/lista de elementos está cadastrado e no qual os usuários podem ter acesso ao elemento (opcional).

Se não informado, a consulta será feita para todos os estabelecimentos.

Unidade - Código da unidade na qual o aplicativo/lista de elementos está cadastrado e na qual os usuários podem ter acesso ao elemento (opcional).

Se não informado, a consulta será feita para todas as unidades.
