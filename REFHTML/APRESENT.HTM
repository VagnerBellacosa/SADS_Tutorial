Apresentação

Este manual se destina a profissionais que atuam como analistas de sistemas, de suporte técnico, como administradores de dados, de banco de dados ou como programadores e que utilizam o SADS como ferramenta para o desenvolvimento de seu trabalho.

O objetivo deste documento é fornecer detalhes técnicos e aprofundamentos teóricos sobre o funcionamento do SADS, de modo a tornar mais claro e produtivo o contato com o software.

Nele encontram-se informações e sugestões para uma utilização mais efetiva dos módulos que compõem o SADS.

Este manual é compatível com a versão 4.2 e é composto por sete seções e três apêndices, cujos conteúdos são sintetizados a seguir:

Seção I - Requisitos e conceitos para utilização do SADS

Esta seção explica a padronização utilizada pelo SADS, os procedimentos de habilitação para o seu uso, o processo de navegação, os tipos de telas utilizadas, as regras para preenchimento dos campos, os diferentes níveis de ajuda e os processos de paginação.

Seção II - Elementos do repositório de dados do SADS

Esta seção apresenta os elementos do projeto conceitual, lógico e físico, em ordem alfabética, fornecendo detalhes que facilitam o desenvolvimento de aplicações com a utilização do SADS.

Seção III - Funções de apoio

Esta seção apresenta as transações que apoiam tanto a utilização do SADS como a dos sistemas construídos através dele, em ordem alfabética.

Com relação aos informativos, apresenta as consultas gerais e gerenciais, os relatórios e as referências cruzadas disponíveis.

Com relação aos utilitários, apresenta funções isoladas, como as de duplicação de elementos e resseqüenciamento de estruturas, e também módulos inteiros, como os de navegação, consulta a cancelamentos on-line e integração ET-SADS x SADS.

Seção IV - Apoio à manutenção

Esta seção apresenta listas de procedimentos que facilitam algumas manutenções mais complexas que envolvem várias transações em sistemas construídos através do SADS.

Seção V - Parametrizações do SADS

A utilização do SADS envolve, mesmo que implicitamente, padrões e métodos de trabalho. Alguns destes padrões são flexíveis e podem ser parametrizados de acordo com a conveniência da instalação.

Seção VI - Sistema de Segurança do SADS

Esta seção apresenta todas as funções e características da ferramenta do SADS responsável por garantir o acesso às informações da empresa.

Seção VII - Servidor WEB

Esta seção apresenta o Servidor Web do SADS, que é um módulo que torna possível a utilização de aplicativos desenvolvidos em SADS, sendo executados em uma máquina Unisys Série A, no ambiente Word Wide Web.

Apêndice A - Arquivos residentes nos ambientes de desenvolvimento e produção

Este apêndice apresenta os objetos, libraries, arquivos e jobs para cada módulo que compõe o SADS que, para permitir o processamento do SADS e dos sistemas aplicativos, necessitam estar residentes nos ambientes de desenvolvimento e produção .

Apêndice B - SADS Secundário

Este apêndice apresenta os procedimentos para criação do ambiente secundário do SADS, que tem como objetivo propiciar a existência de dois ambientes SADS semi-independentes em uma mesma máquina.

Como usar este manual

Este manual deve ser utilizado sob demanda, ou seja, para solucionar alguma dúvida específica durante qualquer fase do desenvolvimento de um sistema.

A Seção II, que descreve os elementos suportados pelo repositório de dados do SADS, não está ordenada logicamente segundo as etapas do desenvolvimento de sistemas. Ela obedece a uma seqüência alfabética em relação aos elementos cadastrados no repositório, sendo que, para cada um deles, são apresentados os tópicos: Conceituação, Padronização, Utilização, Cadastramento e Funções de apoio. A Seção III, que descreve as funções de apoio ao sistema, também está ordenada alfabeticamente, de modo a facilitar a localização de seus itens.

O leitor deve orientar a sua leitura a partir da organização deste manual através de índices. O índice geral apresenta toda a distribuição temática do manual. Os índices parciais, que antecedem a cada seção, proporcionam mais rapidez e facilidade na localização de assuntos específicos.

No cabeçalho e no rodapé de cada página são fornecidos os dados de identificação da seção e a numeração progressiva, a fim de tornar mais fácil e rápida a localização da informação desejada. Além destes recursos, centralizados no cabeçalho de cada página da Seção II encontram-se os nomes dos elementos do SADS que estão sendo explicados.

Os assuntos são identificados a partir de padrões gráficos diferentes para títulos, subtítulos, tópicos e subtópicos, alinhados à margem esquerda das páginas.

Os campos das telas de processamento são apresentados em negrito e e os comandos e valores que os campos das telas podem assumir são apresentados em letras maiúsculas.

Finalmente, no texto principal, estão sendo utilizados caracteres itálicos como recurso para identificação de palavras de origem estrangeira.
