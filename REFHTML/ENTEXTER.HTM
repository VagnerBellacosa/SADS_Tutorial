Entidade externa

Conceituação

A entidade externa representa uma fonte e/ou destino das informações, na fronteira do sistema, podendo ser uma pessoa, um departamento, um sistema automatizado, uma organização externa ou similares, desde que se comunique com o sistema, enviando e/ou recebendo dados.

As entidades externas do DFD podem ser definidas no ET-SADS, que é uma ferramenta UPPER-CASE voltada à aplicação da análise de dados e da análise estruturada e, posteriormente, transferidas para o repositório de dados do SADS. O ET-SADS trabalha na filosofia de estação de trabalho, requerendo uma plataforma IBM-PC ou compatível em ambiente MS-DOS.

As informações relativas à transferência de informações do ET-SADS para o SADS podem ser encontradas no "Guia de Referência do DFD" e no "Guia de Integração com o SADS", manuais que acompanham o ET-SADS.

Padronização

O padrão de codificação para identificar entidades externas no repositório de dados do SADS é o seguinte:

"sssZExxxxx" onde:

"sss" é o código albabético que identifica o sistema;

"ZE" é a constante de entidade externa e

"xxxxx" é o identificador alfanumérico da entidade externa.

Utilização

As informações referentes a entidades externas são utilizadas apenas para documentação, pois podem fazer parte dos manuais emitidos pelo SADS, não havendo gerações a partir de suas especificações.

Cadastramento

Para definir uma entidade externa, é necessário que o sistema já esteja cadastrado no repositório de dados do SADS.

Roteiro

Para definir uma entidade externa, é necessário executar apenas a seguinte função:

Definição de entidade externa

CAMPOS DA TELA

ENTIDADE EXTERNA: Código da entidade externa no padrão SADS (sssZExxxxx).

CODIGO EXTERNO - Código externo da entidade externa.

Este código pode ser informado no caso da instalação adotar um padrão de codificação diferente do padrão SADS ou quando houver integração entre o ET-SADS e o SADS. Neste caso, o código externo da entidade externa no SADS conterá o código da entidade externa no ET-SADS.

O código externo é uma informação puramente documental, ou seja, somente será utilizado para identificar, nos relatórios e manuais, a qual elemento externo ao SADS a entidade externa se refere.

NOME - Descrição resumida da entidade externa.

O nome tem o objetivo de identificar e explicar, de maneira reduzida, a que se refere a entidade externa.

DESCRICAO - Descrição detalhada da entidade externa.

Estão disponíveis 99 páginas, com 10 linhas cada, para se efetuar a descrição detalhada da entidade externa.

CONTINUA - Indicador de continuidade de operação.

As possibilidades de preenchimento deste campo são as seguintes:

"S" - Nova tela será aberta para prosseguir descrevendo a entidade externa.

"N" - Última tela da descrição da entidade externa.

OPERACAO - Operação realizada em relação à entidade externa.

As operações possíveis são as seguintes:

"INC" (Inclusão);

"ALT" (Alteração);

"EXC" (Exclusão);

"CON" (Consulta).
