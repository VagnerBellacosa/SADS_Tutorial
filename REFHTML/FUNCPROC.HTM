Função/processo

Conceituação

Função ou processo é um elemento do DFD - Diagrama de Fluxo de Dados - que representa um conjunto lógico de procedimentos, determinando as transformações a serem efetuadas nos dados do sistema, ou seja, representa um trabalho executado em cima dos dados com o objetivo de transformar as entradas em saídas desejadas.

As funções/processos do DFD podem ser definidas no ET-SADS, que é uma ferramenta UPPER-CASE voltada à aplicação da análise de dado+

s e da análise estruturada, e, posteriormente, transferidas para o repositório de dados do SADS. O ET-SADS trabalha na filosofia de estação de trabalho, requerendo uma plataforma IBM-PC ou compatível em ambiente MS-DOS.

As informações relativas à transferência de informações do ET-SADS para o SADS podem ser encontradas no "Guia de Referência do DFD" e no "Guia de Integração com o SADS", manuais que acompanham o ET-SADS.

Padronização

O padrão de codificação para identificar função/processo no repositório de dados do SADS é o seguinte:

"sssHxxxxxx" onde:

"sss" é o código alfabético que identifica o sistema;

"H" é a constante de função/processo e,

"xxxxxx" é o identificador alfanumérico da função/processo.

Utilização

Todas as informações relativas a funções/processos são utilizadas apenas para documentação, pois podem fazer parte dos manuais emitidos pelo SADS, não havendo gerações a partir de suas especificações.

Cadastramento

Para definir função/processo, é necessário que o sistema já esteja cadastrado no repositório de dados do SADS.

Roteiro

Para definir uma função/processo, é necessário executar apenas a transação de "Definição de função".

Definição de função

CAMPOS DA TELA

FUNCAO - Código da função/processo no padrão SADS (sssHxxxxxx).

NOME - Descrição resumida da função/processo.

O nome tem o objetivo de identificar e explicar, de maneira reduzida, a que se refere a função/processo.

FUNCAO NIVEL SUPERIOR - Código da função que, no diagrama hierárquico funcional, está situada no nível imediatamente superior à que está sendo definida.

Este campo não deve ser informado para o primeiro nível funcional.

CODIGO ESTRUTURADO - Código numérico estruturado utilizado para indicação da posição hierárquica da função/processo no DHF (Diagrama hierárquico funcional).
Ao listar as descrições das funções/processos do sistema, elas serão impressas pela ordem do código estruturado.

DESCRICAO - Descrição detalhada da função/processo.

Estão disponíveis 99 páginas, com 10 linhas cada, para se efetuar a descrição detalhada da função/processo.

CONTINUA - Indicador de continuidade de operação.

As possibilidades de preenchimento deste campo são as seguintes:

"S" - Nova tela será aberta para prosseguir descrevendo a função/processo.

"N" - Última tela da descrição da função/processo.

OPERACAO - Operação efetuada em relação à função/processo.

As operações possíveis são as seguintes:

"INC" (Inclusão);

"ALT" (Alteração);

"EXC" (Exclusão);

"CON" (Consulta).

Funções de apoio

Específicas de função/processo

O SADS dispõe da transação de "Relacionamento função x transações", aplicável à função/processo. Esta transação está descrita a seguir:

Relacionamento função x transações

O SADS permite, através do relacionamento de função x transações, definir o conjunto de transações que automatiza uma determinada função/processo.

Para estabelecer esse relacionamento, não é necessário que as transações estejam cadastradas no repositório de dados do SADS, pois elas serão definidas somente no modelo físico.

CAMPOS DA TELA

FUNCAO - Código da função/processo no padrão SADS (sssHxxxxxx).

TRANSACOES - Códigos das transações (máximo de dez) que automatizam a função.

As transações devem ser informadas no padrão SADS (sssUmmnn) da seguinte forma:

"sssUmmnn" onde:

"sss" é o sistema;

"U" é a constante que identifica transação;

"mm" é o menu e

"nn" é o número da transação.

Genéricas aplicáveis às funções/processos

Utilitários:

Descrição de elemento;

Exportação de descrição de elemento;

Importação de descrição de elemento.

Consultas:

Elementos por sistema;

Descrição de elementos.

Informativos Impressos:

Lista de elementos por sistema;

Lista de descrições de elementos.

Todos estes utilitários, consultas e informativos impressos encontram-se detalhadamente descritos na Seção III, "Funções de apoio", deste manual.
