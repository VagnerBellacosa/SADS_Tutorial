Job

Conceituação

Job é uma rotina em linguagem WFL através da qual é possível executar os objetos dos programas batch do sistema.

Padronização

O padrão de codificação adotado pelo SADS para a identificação de jobs é o seguinte:

"sssIxxxx" onde:

"sss" é código alfabético que identifica o sistema;

"I" é a constante de job e

"xxxx" é o identificador alfanumérico do job.

Utilização

O cadastramento de job no SADS é puramente documental e tem o objetivo de relacionar os programas que serão executados através dele, de modo a manter uma referência cruzada.

Cadastramento

Para efetuar o cadastramento de job, é necessário que o sistema ao qual ele pertence e os programas que a ele serão relacionados estejam cadastrados no repositório de dados do SADS.

Roteiro

Encontra-se disponível para esse elemento apenas a seguinte função:

-Definição de job.

Definição de job

CAMPOS DA TELA

JOB - Código do job no padrão SADS (sssJxxxx).

CODIGO EXTERNO - Código diferente do padrão do SADS.

O código externo é geralmente utilizado para especificar códigos de jobs no padrão do software SPO - Sistema de Controle de Processamento.

NOME Descrição resumida do job.

O nome tem por objetivo identificar e explicar resumidamente a que o job se refere.

PROGRAMAS QUE COMPOEM O JOB - Código dos programas que compõem o job, no padrão SADS (sssPxx).

Podem ser informados até 20 (vinte) programas batch por job. Estes programas têm de estar cadastrados no repositório de dados do SADS. Os programas são exibidos na ordem em que forem informados, e não em ordem alfabética.

OPERACAO - Operação efetuada sobre o job.

As operações possíveis são as seguintes:

"INC" (Inclusão);

"ALT" (Alteração);

"EXC" (Exclusão);

"CON" (Consulta).
