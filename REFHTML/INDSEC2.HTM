Seção II

Elementos do repositório de dados do SADS

Esta seção apresenta os elementos do projeto conceitual, lógico e físico, fornecendo detalhes que facilitam o desenvolvimento de aplicações com a utilização do SADS.

Nesta seção, os elementos cadastrados no repositório de dados do SADS foram ordenados alfabeticamente, portanto, não estão dispostos em seqüencia lógica de utilização segundo as etapas do desenvolvimento de sistemas.
