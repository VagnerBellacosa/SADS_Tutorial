Depósito de dados

Conceituação

Depósito de dados é, no Diagrama de Fluxo de Dados - DFD, o local de armazenamento de informações de um sistema.

Os depósitos de dados do DFD podem ser definidos no ET-SADS, que é uma ferramenta UPPER-CASE voltada à aplicação da análise de dados e da análise estruturada e, posteriormente, transferidos para o repositório de dados do SADS. O ET-SADS trabalha na filosofia de estação de trabalho, requerendo uma plataforma IBM-PC ou compatível em ambiente MS-DOS.

As informações relativas à transferência de informações do ET-SADS para o SADS podem ser consultadas no "Guia de Referência do DFD" e no "Guia de Integração com o SADS", manuais que acompanham o software.

Padronização

O padrão de codificação para identificar depósitos de dados no repositório de dados do SADS é o seguinte:

"sssXDxxxxx" onde:

"sss" é o código alfabético que identifica o sistema;

"ZD" é a constante de depósito de dados e

"xxxxx" é o identificador alfanumérico do depósito.

Utilização

As informações referentes aos depósitos de dados são utilizadas apenas para documentação, pois podem fazer parte dos manuais emitidos pelo SADS, não havendo gerações a partir de suas especificações.

Cadastramento

O processo de definição de um depósito de dados só é possível a partir do cadastramento do sistema no repositório de dados do SADS.

CAMPOS DA TELA

DEPOSITO DE DADOS - Código do depósito de dados no padrão SADS (sssZDxxxxx).

CODIGO EXTERNO - Código externo do depósito de dados.

Este código pode ser informado no caso da instalação adotar um padrão de codificação diferente do padrão SADS ou quando houver integração entre o ET-SADS e o SADS. Neste caso, o código externo do depósito de dados no SADS conterá o código do depósito de dados no ET-SADS.

O código externo é uma informação puramente documental, ou seja, somente será utilizado para identificar, nos relatórios e manuais, a qual elemento externo ao SADS o depósito de dados se refere.

NOME - Descrição resumida do depósito de dados.

O nome tem o objetivo de identificar e explicar, de maneira reduzida, a que se refere o depósito de dados.

DESCRICAO - Descrição detalhada do depósito de dados.

Estão disponíveis 99 páginas, com 10 linhas cada, para se efetuar a descrição detalhada do depósito de dados.

CONTINUA - Indicador de continuidade de operação.

As possibilidades de preenchimento deste campo são as seguintes:

"S" - Nova tela será aberta para prosseguir descrevendo o depósito de dados;

"N" - Última tela da descrição do depósito de dados.

OPERACAO - Operação efetuada em relação ao depósito de dados.

As operações possíveis são as seguintes:

"INC" (Inclusão);

"ALT" (Alteração);

"EXC" (Exclusão);

"CON" (Consulta).
